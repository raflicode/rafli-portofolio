# Rafli Portfolio

Personal portfolio website built with Vue 3, Tailwind CSS, GSAP, and vue-i18n.

## 🚀 Tech Stack

- Vue 3
- Vite
- Tailwind CSS
- GSAP
- Lucide Icons
- vue-i18n

## ✨ Features

- Responsive design
- Dark and Light mode
- Language switcher (Indonesia & English)
- Smooth animations
- Interactive UI
- Download CV
- Contact section

## 📦 Installation

```bash
npm install
npm run dev
